## 📖 על הפרויקט

אפליקציה המציגה **18 תרחישים נפוצים** שמפתחות (ומפתחים) נתקלים בהם –
ומדגימה את ההבדל בין פרומפט "עצלני" לפרומפט מקצועי.

הדשבורד כולל:
- 🔍 **חיפוש תרחישים** – מצאי בדיוק מה שרלוונטי אליך
- ❌ **גישה עצלנית** – מה לא לעשות ולמה
- ✅ **גישה מקצועית** – פרומפט מותאם ל-IDE עם תשובה צפויה
- 📋 **מתכון 4 שלבים** – סמני → שאלי ממוקד → קראי → למה?

## 🚀 התקנה והרצה

```bash
# שכפול הפרויקט
git clone https://github.com/<your-username>/how-to-prompt.git
cd how-to-prompt

# התקנת תלויות
npm install

# הרצה בסביבת פיתוח
npm run dev
```

האפליקציה תעלה בכתובת `http://localhost:5173`
